# Broad Challenge

Hello!

I used Python (3.9) and the `requests` package to complete the challenge.

## Running 

```bash
python3 challenge.py --apikey <APIKEY>
```

`--apikey` is optional.

If you want to run a single question, you can do, e.g.:

```bash
python3 challenge.py --apikey <APIKEY> --q q1
```
